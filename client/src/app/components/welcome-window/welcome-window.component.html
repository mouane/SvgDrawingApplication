PolyDessin
Bienvenue sur PolyDessin. Ce logiciel vous permet de créer des chef d'oeuvres et de les enregistrer sur notre serveur.
Sur le coté gauche se trouve la zone des outils de dessin. Il vous est possible de selectionner de multiples outils pour ensuite créer un dessin dans le zone de production a droite.
Pour plus d'informations, veuillez visiter le guide d'utilisation, qui est accessible par le bouton à gauche en bas de page.
