Guide d'utilisation - PolyDessin

Créer un dessin

Pour créer votre premier dessin, suivez les étapes suivantes:

  1-Appuyez sur le premier icone '+' qui se situe dans le coin supérieur gauche de l'écran

  2-Choisissez la hauteur et la largeur de votre nouveau dessin (supérieur à 0px, inférieur à 5000px)

  3-Choisissez la couleur de l'arrière plan (soit en hexa, soit en RGBA)

  4-Appuyez sur Ok!

Il est important de noter que vous ne pouvez utiliser aucun outil sans avoir créé un nouveau dessin

Outils offerts

PolyDessin vous permet d'utiliser plusieurs outils que vous pouvez utiliser pour créer vos chef d'oeuvres. Le nom de chaque outil est obtenu en passant la souris sur un outil. Ces derniers sont expliqués ci-dessous, commençant par le haut de la barre latérale à gauche de l'écran:


Outils de tracé

Crayon

L'outil crayon vous permet de dessiner des traits. Vous pouvez changer l'épaisseur de vos traits ainsi que la couleur en choisissant une couleur primaire

Stylo

L'outil stylo vous permet de dessiner des traits. La vitesse du tracé est inversement proportionnelle avec son épaisseur. Vous pouvez changer la couleur de vos traits en choisissant une couleur primaire

Pinceau

L'outil pinceau vous permet de dessiner des textures (cinq options disponibles). Vous pouvez changer l'épaisseur de vos textures ainsi que la couleur en choisissant une couleur primaire

Applicateur de couleur

L'applicateur de couleur permet de changer la couleur d’un objet (rectangle, ellipse, cercle, etc.) en cliquant dessus. Un clic avec le bouton gauche changera la couleur pour la couleur principale. Un clic avec le bouton droit changera la couleur de bordure pour la couleur secondaire.

Couleur de fond

Il est possible de changer la couleur de fond de votre dessin à tout moment en utilisant l'outil de couleur ce trouvant sur la barre latérale.

Ligne

Cet outil vous permet de choisir une taille et un type de ligne voulu pour ensuite pouvoir créer des lignes a l'aide du clique gauche de la souris. Il est possible d'utiliser les raccourcis : Escape pour annuler la ligne en cours de création et Backspace pour annuler le segment de ligne créé le plus récent. Pour arrêter de faire des segments de ligne, il ne suffit que de double-cliquer à la fin de notre ligne.

Pipette

La pipette permet de saisir et d'assigner des couleurs. Un clic gauche assigne la couleur selectionnée à la couleur primaire. Un clic droit assigne la couleur selectionnée à la couleur secondaire. Pour rendre ce changement de couleur officiel, il suffit de cliquer sur le bouton "Appliquer" se trouvant sur la barre latérale lorsque l'outil pipette est selectionné.

Rectangle

Cet outil vous permet de dessiner des rectangles. Vous pouvez choisir de dessiner soit seulement le contour d'un rectangle, soit le plein, soit les deux. Il est aussi possible pour vous de choisir l'épaisseur du contour d'un rectangle. Vous pouvez aussi definir la couleur principale (le remplissage) ainsi que la couleur secondaire (le contour).

Ellipse

Cet outil vous permet de dessiner des ellipses. Tout comme le rectangle, vous pouvez choisir de dessiner soit le contour de l'ellipse, soit le plein, soit les deux. Il est aussi possible de définir l'épaisseur du contour de l'ellipse. Un glisser déposer permet de dessiner une ellipse occupant la plus grande aire dans un rectangle. Pour dessiner un cercle, il suffit d'appuyer dur la touche "shift".

Polygone

Cet outil vous permet de dessiner des polygones. Le nombre de côtés souhaités (entre 3 et 12), ainsi que l'épaisseur du contour des polygones peuvent être choisis sur la barre latérale. Tout comme le rectangle, vous pouvez choisir de dessiner soit le contour de l'ellipse, soit le plein, soit les deux. Un glisser déposer permet de dessiner un polygone occupant la plus grande aire dans un rectangle.

Grille

Cet outil vous permet d'afficher une transparente en fond d'écran. Il est possible d'incrémenter la taille des carrés de la grille avec le raccourci + ou de réduire celle-ci avec le raccourci -

Étampes

Cet outil vous permet de choisir une image, son échelle et son angle de rotation par rapport a l'image échantillon de base et ensuit insérer cette image dans la zone de dessin. Il est possible d'utiliser le raccourci : Alt + cran de roulette pour augmenter l'angle de rotation d'une étampe. Au niveau du fonctionnement, il faut simplement sélectionner l'image voulu en cliquant sur celle-ci, choisir ses paramètres ou utiliser ceux de base, et ensuite cliquer sur l'endoit de la zone de dessin pour y faire apparaitre l'étampe.

Sauvegarder un fichier

Il est possible de sauvegarder votre dessin de deux manières différentes

  1-Sauvegarde serveur: cet options vous permet de sauvegarder le dessin courant dans la base de données de Polydessin. Il n'est pas possible d'ouvrir l'option sauvegarder s'il n'y a pas de dessin en cours.

  2-Sauvegarde local: cet options vous permet de sauvegarder le dessin courant localement sur votre ordinateur. Le fichier se trouvera dans le dossier téléchargé par défaut. Il n'est pas possible d'ouvrir l'option sauvegarder s'il n'y a pas de dessin en cours.

Ouvrir un fichier

Il est possible d'ouvrir un ancien dessin qui a été sauvegardé de deux manières différentes

  1-Ouverture du serveur: cet options vous permet d'ouvrir un dessin qui a auparavent été sauvegardé dans la base de données de polydessin. Il est possible de filtrer à travers les dessins sauvegardés à l'aide des étiquettes.

  2-Ouverture local: cet options vous permet d'ouvrir un dessin qui a auparavent été sauvegardé localement.

Exporter le dessin

Cette fonctionnalité permet d’exporter un dessin localement sous les formats suivants : bmp, png, jpg et svg.
